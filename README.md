# tvdb_client

`tvdb_client` is a rubygem to interface with The TVDB's REST API.

## Installation

```ruby
gem install 'tvdb_client'
```

Or include it in your Gemfile:

```ruby
source "https://rubygems.org"

gem 'tvdb_client'
```

and then install via bundler:

```ruby
bundle install
```
